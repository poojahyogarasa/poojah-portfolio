## ⚙️ Getting Started

### Clone Repository

```bash
git clone https://github.com/poojahyogarasa/poojah-portfolio.git
```

### Navigate to Project

```bash
cd poojah-portfolio
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
